## Technical Approach

### Feature Engineering

* Combined odorant descriptors with EEG response text
* NLP vectorization using TF-IDF
* Cognitive/emotional brain states encoded as classification targets

### Machine Learning Models

* Logistic Regression (baseline interpretability)
* XGBoost (non-linear performance)
* TensorFlow Neural Network (scalable deep learning)

### Explainability & Ethics

* SHAP used for interpretability
* Transparent decision reasoning
* Responsible AI safeguards embedded in design

---

## Jury-Safe Definition of Agentic AI

This project does NOT implement autonomous AI agents.

Instead, agentic AI is used as:

> A decision-support intelligence layer that dynamically adapts recommendations based on user input and contextual signals, while maintaining human oversight.

Key safeguards:

* Human-in-the-loop design
* No psychological manipulation
* No biometric storage without consent
* AI provides recommendations, not decisions
