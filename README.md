## Overview

**NeuroOlfactive Intelligence** is a conceptual web application prototype designed to demonstrate how **neuro-olfactive AI**, **biotechnology**, and **immersive digital experiences** can redefine luxury fragrance discovery.

Unlike traditional e-commerce, this platform uses a simulated "biotech" analysis to decode a user's emotional and neurological state, generating a unique, personalized scent profile that bypasses cognitive bias to trigger pure emotion.

## Getting Started

To run this prototype locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/neuro-olfactive.git
    cd neuro-olfactive
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```

## Conceptual Background

This project was designed as a "jury-ready" prototype for an innovation competition. It adheres to strict design guidelines:
*   **Neutral Palette**: Ivory, Charcoal, Muted Gold.
*   **No Gamification**: Serious, scientific, and aspirational tone.
*   **Responsible AI**: Transparent "Explainability" metrics (SHAP-inspired).
